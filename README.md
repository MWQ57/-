# 智能衣橱

个人智能衣橱 Web 应用，支持衣物数字化管理、AI 智能搭配、天气穿搭推荐、场景穿搭推荐和自由 DIY 搭配画布。

## 功能

- **衣物数字化**：拍照上传 / 批量相册上传，自动抠图透明底存档，AI 智能识别品类、颜色、面料、风格、季节、版型
- **衣橱管理**：智能分类、多维度筛选搜索、状态标注（常穿/闲置/待清洗）、本地持久化存储
- **AI 智能搭配**：选择单品自动生成多套成套穿搭方案
- **天气穿搭**：对接 Open-Meteo 天气 API，支持城市搜索切换，根据温度、晴雨、风力、紫外线推荐穿搭
- **场景穿搭**：通勤办公、日常休闲、社交约会、正式场合、运动户外、季节节日
- **DIY 画布**：拖拽摆放、层级调节、智能提示、方案保存与对比

## 快速开始

```bash
npm install
npm run dev
```

浏览器访问 http://localhost:5173

## 构建部署

```bash
npm run build
npm run preview
```

## 技术栈

- React 19 + TypeScript + Vite
- Tailwind CSS 4
- Zustand 状态管理
- Open-Meteo 天气 API（免费，无需 API Key）
- localStorage 本地数据持久化

## 说明

- AI 识别基于主体轮廓分析（宽高比、质量分布）+ 色彩提取 + 规则推断，上传后可人工微调
- 抠图使用 @imgly/background-removal 本地模型，首次使用需下载模型（约 40MB）
- 数据保存在浏览器本地，清除缓存会丢失数据
- 天气功能需要网络连接和定位权限（拒绝定位时默认使用北京）
